# Ecommerce Growth Engine

Comprehensive E-commerce Analytics built with Python & Streamlit to optimize marketing ROI, conversion funnels, product perfromance, and customer loyalty through data-driven insights.

  **Project Status:** This projeic is **Under Development**. Features and documentaions are being updated in real-time.

## Project Overview:

Dataset contains data of an ecommerce retailer. This project aims to transform raw database exports into actionable business intelligence using a modern data stack.

The analysis is structured into four major domains:

  **1. Marketing Channel Efficieny:** Optimizing spend across Gsearch, Bsearch, and social channels.
  
  **2. Conversion Funnel Optimization:** Identifying user drop-off points from landing pages to checkout.
  
  **3. Product Performance:** Measuring the impact of products and monitoring refund rates.
  
  **4. Customer Lifecycle:** Analyzing repeat purchase behavior and long-term user value.
